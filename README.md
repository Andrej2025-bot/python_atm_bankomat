# Python ATM Bankomat

Jednoduchý bankomat naprogramovaný v jazyku Python.
Projekt slúži ako tréning základov programovania.

## Funkcie
- zadanie PIN kódu
- obmedzený počet pokusov
- zobrazenie zostatku
- vklad peňazí
- výber peňazí
- ukončenie programu

## Použité koncepty
- while cyklus
- if / elif / else podmienky
- práca s premennými
- vstup od používateľa (input)

## Spustenie programu
Program spustíš príkazom:

python test.py
